## united-we-stand integration

This repository uses the **united-we-stand** spec-driven AI workflow framework.

**Before making any changes, you must:**

1. Read `AGENTS.md` at the repository root.
2. Read the applicable agent files in `.united-we-stand/agents/`.
3. Read the branch-specific spec files in `.united-we-stand/spec-driven/<current-branch>/`.
4. Follow the staged framework process described in AGENTS.md.
